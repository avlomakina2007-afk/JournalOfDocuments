# Проект "Журнал входящих документов" для ООО "РЭМ"

## Описание проекта
WPF-приложение для административного отдела, позволяющее вести учёт входящих документов (регистрационный номер, дата, отправитель, тема, исполнитель).

## Технологии
- .NET Framework 4.8
- WPF (XAML)
- C# 7.3
- Newtonsoft.Json для хранения данных

## Функциональность
- Просмотр списка документов в DataGrid
- Добавление нового документа
- Редактирование выбранного документа
- Удаление документа с подтверждением
- Автосохранение в JSON-файл
- Фильтрация по отправителю (добавлено в ветке feature/add-filter)

## Структура репозитория
- MainWindow.xaml – разметка главного окна
- MainWindow.xaml.cs – логика приложения
- Document.cs – модель данных
- documents.json – файл хранения (создаётся автоматически)

## Инструкция по запуску
1. Установить Visual Studio 2019/2022 с поддержкой .NET Framework 4.8.
2. Открыть решение JournalOfDocuments.sln.
3. Восстановить NuGet-пакет Newtonsoft.Json (если не восстановился).
4. Нажать F5.
